# Smart QR Generator and Scanner

A Python-based Smart QR Generator and Scanner project.

## Features
- Generate QR Codes
- Scan QR Codes
- GUI Support
- Image Saving

## Technologies Used
- Python
- OpenCV
- QRCode
- Pillow
- Tkinter

## How to Run

### Install Libraries
pip install qrcode pillow opencv-python

### Run Generator
python app.py

### Run Scanner
python scanner.py
